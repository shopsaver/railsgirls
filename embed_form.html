XYZ Corporation Bug Bounty Program

Welcome to the official bug bounty program for XYZ Corporation. Help us improve our security and earn rewards!

Program Overview

We are offering a bug bounty program to identify vulnerabilities in our system. If you find any security issues, please report them using the form below. We appreciate your help in making our product more secure!

Scope

  • Website - Any critical vulnerabilities such as XSS, SQL Injection, etc.
  • API - Issues related to authentication, authorization, and data access.
  • Mobile Apps - Android and iOS applications (please specify the platform when submitting).
  • Smart Contracts - Issues related to Ethereum-based smart contracts used in our platform.

Program Rules

  • No social engineering or phishing attacks.
  • Only report issues that are not publicly disclosed.
  • Do not disrupt service during testing.
  • Submit complete and clear reports with all necessary details for reproduction.

Rewards

Rewards will be determined based on the severity of the vulnerability found. The following is a guideline:

  • Critical: $5,000 - $10,000
  • High: $1,000 - $4,999
  • Medium: $500 - $999
  • Low: $50 - $499
